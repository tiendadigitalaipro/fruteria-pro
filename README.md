# 🍎 FruteriaPro — Sistema POS para Fruterías

Sistema completo de punto de venta para fruterías y verdulerías. 100% offline, sin instalación.

## ✨ Características
- 74 productos en 4 categorías (Frutas, Verduras, Tubérculos, Granos)
- POS completo con carrito, descuentos y vuelto
- 8 métodos de pago (Efectivo $, Bs, Pago Móvil, Zelle, Binance...)
- Inventario con alertas de stock mínimo
- Historial de ventas con exportar CSV
- Reportes y top 10 productos
- Tasa Bs/$ configurable
- Moneda dual: Dólares ($) + Bolívares (Bs)
- 100% offline — sin Firebase, sin internet requerido

## 🚀 Cómo usar
1. Doble clic en `ABRIR-FRUTERIA.bat`
2. O abre `fruteria-pro.html` directamente en Chrome/Edge

## 🛡️ Datos y Seguridad
- Todos los datos se guardan localmente en el navegador (localStorage)
- Exportar backup regularmente desde Configuración → Exportar JSON
- Sin conexión a internet requerida
